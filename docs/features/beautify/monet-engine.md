# 莫奈引擎

> 为微信的部分组件启用动态壁纸取色 \[需 SDK >= 31\]

## 类别

界面美化

## 类型

可开关 & 可点击

## 描述

利用 Android 12+ (API 31) 的 Material You 动态取色系统, 让微信的部分界面元素跟随系统壁纸颜色自动换色。受影响的组件包括:

- **开关按钮** (MMSwitchBtn) —— 启用状态颜色跟随系统主色
- **文本框光标与选择手柄** —— 应用系统主色
- **按钮背景** —— 应用系统辅助色
- **背景颜色** —— 默认灰色背景替换为系统主色
- **Paint 颜色绘制** —— 匹配系统主色

如果动态壁纸取色没有生效, 说明系统 Android 版本过低 (SDK < 31)。

## 使用方法

在模块设置中启用
