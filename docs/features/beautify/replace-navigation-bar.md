# 美化首页底部导航栏

> 将首页底部导航栏替换为 Material Design 或 Backdrop 风格

## 类别

界面美化

## 类型

可开关 & 可点击

## 描述

将微信首页底部的原生导航栏替换为全新的 Material Design 3 风格导航栏, 提供两种样式选择:

- **标准 M3 导航栏** —— Material Design 3 底部导航栏, 带有滑动渐变动画效果, 图标跟随页面切换平滑过渡
- **悬浮底栏 (Floating Bottom Bar)** —— 悬浮式底部导航栏, 可额外启用液态玻璃 (Backdrop) 背景模糊效果

两种样式均支持未读消息角标显示 (主页 Tab)。

## 使用方法

在模块设置中启用, 可点击以选择「使用悬浮底栏」和「启用液态玻璃效果」
