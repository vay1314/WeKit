# 脚本引擎

> 执行 JavaScript 脚本

## 类别

脚本

## 类型

可开关

## 描述

启用后将自动加载 `<模块数据>/scripts/` 目录下的所有 `.js` 脚本文件。脚本引擎为微信提供了 JavaScript 扩展能力, 支持通过脚本监听消息、拦截和修改网络请求与响应, 详见 [脚本引擎 API 参考](js-api-reference.md)。

支持的事件函数:

- `onMessage()` — 收到消息时触发
- `onRequest()` — 发起网络请求时触发
- `onResponse()` — 收到网络响应时触发

## 使用方法

在模块设置中启用, 并将 `.js` 脚本文件放入 `<模块数据>/scripts/` 目录, 模块会在下次微信启动时自动加载并执行
