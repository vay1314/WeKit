# 聊天输入栏增强

> 为聊天输入栏添加更多功能。1. 在聊天界面长按「发送」或「加号菜单」按钮打开菜单: 「发送卡片消息」「@所有人」。2. 长按「语音」按钮发送自定义语音文件 (SILK 或 MP3)

## 类别

聊天

## 类型

可开关

## 描述

1. **长按「发送」或「加号菜单」按钮** 打开菜单:
   - **发送语音文件** — 打开文件选择器, 选择 AMR 或 MP3 音频文件发送为语音消息
   - **发送卡片消息** — 将输入框内容作为 XML AppMsg 发送 (即微信分享卡片、链接预览等使用的消息格式, 需输入 XML 格式内容)
   - **@所有人** — 向群内所有成员发送 @ 通知 (仅群聊可用)
2. **长按「语音」按钮** 打开文件选择器: 选择 AMR 或 MP3 音频文件发送为语音消息

## 使用方法

在模块设置中启用
