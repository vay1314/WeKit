# 对话分组

> 向主页顶部添加 Tab 栏, 按条件快速筛选对话。建议同时启用「界面美化/隐藏主页下滑「最近」页」

## 类别

聊天

## 类型

可开关

## 描述

在微信主页顶部添加包含 5 个 Tab 的栏:

| Tab | 筛选条件 |
|-----|---------|
| 全部 | 显示所有对话 |
| 未读 | 仅显示有未读消息的对话 |
| 群聊 | 仅显示群聊 (@chatroom)  |
| 好友 | 仅显示个人好友 (wxid_%)  |
| 公众号 | 仅显示公众号 (gh_%)  |

Tab 筛选会与「隐藏联系人」联动, 被隐藏的联系人不会出现在筛选结果中。

建议同时启用「界面美化/隐藏主页下滑「最近」页」以获得更好的使用体验。

## 使用方法

在模块设置中启用
